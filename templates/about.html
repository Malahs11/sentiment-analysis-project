AI-Powered Sentiment Analysis Tool

Project Objective

This project analyzes user reviews and predicts sentiments such as positive, negative, and neutral using Natural Language Processing and Machine Learning techniques.

Features

  • Sentiment Prediction
  • Emotion Detection
  • Pie Chart Analytics
  • Word Cloud Visualization
  • CSV Bulk Analysis
  • User Login System

Technologies Used

  • Python
  • Flask
  • Machine Learning
  • NLP
  • Scikit-learn
  • Matplotlib
  • WordCloud
